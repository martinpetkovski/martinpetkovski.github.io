      Shortcuts

      • Ctrl + O: Open folder
      • Ctrl + S: Save all files
      • Ctrl + PageDown: Next file
      • Ctrl + PageUp: Previous file
      • Alt + O: Switch header/source
      • Shift + Alt + S: Open symbol search
      • Ctrl + Shift + F: Find in files
      • Alt + G: Go to definition/declaration
      • Alt + ArrowRight/Left: Move to next/previous subword
      • Ctrl + G: Go to line
      • Ctrl + R: Rename symbol
      • Ctrl + H: Show this help menu
      • Ctrl + -: Go to previous cursor location
      • Ctrl + Shift + -: Go to next cursor location
