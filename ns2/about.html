Hey there! Welcome to the About page for the NS2 Editor.

This bad boy is my web-based tool for crafting and tweaking interactive stories or games in the NS2 format. It's got a slick visual interface so you don't have to code everything from scratch, but if you're feeling adventurous, you can totally mess with the raw scripts too. All powered by the NS2 engine under the hood.

Versions

  • NS2 Editor: 2.0
  • NS2 Engine: 2.0

I, Martin Petkovski, put this together in 2025.

Fun note

This whole thing kicked off as my master's thesis way back in 2022.
