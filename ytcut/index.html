ytcut (YouTube Cutter)

Windows console tool that uses yt-dlp + ffmpeg to download and trim a segment.

Downloads

  • ytcut-win64.zip (includes ytcut + yt-dlp + ffmpeg)

Requirements

  • Windows + internet access
  • No additional installs (everything is bundled)

Usage

  • Interactive: ytcut.exe
  • CLI (full download): ytcut.exe --url "https://www.youtube.com/watch?v=..."
  • CLI (trim): ytcut.exe --url "https://www.youtube.com/watch?v=..." --start 00:01:23 --end 00:02:10 --out clip.mp4
  • Output: if --out is omitted, writes out.mp4 next to ytcut.exe (relative paths are also relative to that folder)
